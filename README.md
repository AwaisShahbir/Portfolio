# Awais's Portfolio

A modern, highly animated, and fully responsive personal portfolio website built to showcase my projects, skills, and professional experience.

## ✨ Features

- **Modern UI/UX**: Dark, futuristic aesthetic with glassmorphism effects.
- **Advanced Animations**: Smooth page transitions and "antigravity" floating elements powered by Framer Motion.
- **Dynamic Routing & Components**: Well-structured React components for improved maintainability.
- **Fully Responsive**: Carefully tailored layouts for everything from mobile devices to large desktop monitors.

## 🛠️ Technology Stack

- **Core**: [React 18](https://react.dev/)
- **Tooling**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Tailored Color Palette & Modern Typography)

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AwaisShahbir/Portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Portfolio
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Running Locally

To fire up a local development server, run:
```bash
npm run dev
```
Then open your browser to `http://localhost:5173`.

## 📦 Build for Production

To generate a production-ready build, run:
```bash
npm run build
```
The optimized bundle will be put into the `dist` folder.
